Forkin' Spoon

No Nonsense Recipes

Forkin' Spoon is a personal project by myself to collect a set of recipes. Within these recipes i'd like to have them eventually connected to a back-end database. There I can have another table for ingredients, ratings, notes, and other things recipe related. I'd also like to keep track of when I have cooked these recipes. This project is also something I am taking on to relearn HTML5, CSS3, and Javascript. From there I'd like to take this project and make it a full-fledged web application done in React. This will be for my professional portfolio and for my own personal usage.
